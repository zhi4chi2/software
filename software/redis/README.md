版本 4.0.0(redis-4.0.0.tar.gz)


# install
https://redis.io/download


```bash
wget http://download.redis.io/releases/redis-4.0.0.tar.gz
tar xzf redis-4.0.0.tar.gz
cd redis-4.0.0
make
cd src
./redis-server
./redis-cli
```
